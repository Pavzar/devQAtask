# Automation task (Java + Selenium + JUnit)

## Tech stack / dependencies

- Java 25
- Chrome browser (tests use `ChromeDriver`)
- Selenium 4.41.0
- Junit 6.0.3
- Maven 

## Test output

After each test results are written to:

`testResults/test_<testMethodName>_<timestamp>.txt`

The same output is also printed to the console.
